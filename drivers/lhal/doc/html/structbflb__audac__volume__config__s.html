AUDAC volume configuration structure

参数
mute_ramp_enAUDAC mute mode en, use true or false
mute_up_ramp_rateAUDAC mute up ramp rate, valid when mute_ramp_en is true, use audac volume ramp rate
mute_down_ramp_rateAUDAC mute down ramp rate, valid when mute_ramp_en is true, use audac volume ramp rate
volume_update_modeAUDAC volume update mode, use audac volume ramp rate
volume_ramp_rateAUDAC volume ramp rate, valid when volume_update_mode is not AUDAC_VOLUME_UPDATE_MODE_FORCE, use audac volume ramp rate
volume_zero_cross_timeoutAUDAC volume update zero cross timeout period, valid only in AUDAC_VOLUME_UPDATE_MODE_RAMP_ZERO_CROSSING mode
